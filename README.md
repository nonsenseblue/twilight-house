# Twilight House (たそがれ荘)

A pixel art room explorer where you live with 4 characters in a cozy Japanese house.

**Play now:** https://twilight-house.vercel.app

## Features

- **5 rooms to explore** - Tatami room, Western room, Engawa (veranda), Kitchen, Bathroom
- **4 characters** - Choose your character, meet the others as NPCs
- **Talk to characters & furniture** - Walk up and press SPACE to interact
- **Retro TV** - Watch news, drama, and anime on a pixel art CRT television
- **Runner mini-game** - Play "GM Butsu Runner" on the TV (Phaser 3)
- **Minimap** - Toggle with M key or click the MAP header

## Controls

| Key | Action |
|---|---|
| Arrow keys / WASD | Move |
| SPACE / Enter | Interact (doors, TV, NPCs, furniture) |
| M | Toggle minimap |
| ESC | Exit TV mode |

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** (build tool)
- **Phaser 3** (mini-game)
- **Canvas 2D** (room rendering)
